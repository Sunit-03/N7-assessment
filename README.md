# N7 — Modern Banking Landing Page

A pixel-perfect landing page for N7 Digital Banking, built with Next.js 16, React 19, and Tailwind CSS 4.

## Prerequisites

- Node.js ≥ 20
- npm ≥ 10

## Setup

```bash
# Clone the repo
git clone https://github.com/Sunit-03/N7-assessment.git
cd N7-assessment

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- **Framework**: Next.js 16 (App Router, Turbopack)
- **UI**: React 19, Tailwind CSS 4
- **Fonts**: Archivo, Chivo Mono, Roboto (via `next/font`)
- **Icons**: Lucide React
